Про BMS Monitor
Система моніторингу акумуляторних батарей на базі ESP32 та JBD BMS

Що це?
BMS Monitor - це веб-додаток для моніторингу батарейних систем управління (BMS) через ESP32 або пряме Bluetooth підключення. Система дозволяє відстежувати напругу, струм, температуру, стан заряду та інші параметри акумуляторних батарей в реальному часі.

Можливості
Моніторинг в реальному часі через ESP32 або Bluetooth
Підтримка паралельного підключення декількох батарей
Відображення напруги кожної комірки
Графіки історії напруги, струму та заряду
Статус захисту (перенапруга, перезаряд, перегрів)
Управління пристроями через групи
Демо режим для тестування
Підтримка української та англійської мов
Як підключити BMS?
Спосіб 1: Через ESP32 (Wi-Fi)
1. Перейдіть на сторінку "Налаштування"
2. Введіть дані Wi-Fi мережі (SSID та пароль)
3. Вкажіть Device ID для вашого ESP32 (наприклад: ESP32_001)
4. Натисніть "Сканувати Bluetooth" та оберіть ваш JBD BMS
5. Завантажте згенеровану прошивку (.ino файл)
6. Відкрийте файл в Arduino IDE
7. Під'єднайте ESP32-C3 Super Mini через USB
8. Виберіть плату: ESP32C3 Dev Module
9. Завантажте прошивку на ESP32
10. ESP32 автоматично підключиться до BMS і почне надсилати дані
Спосіб 2: Пряме підключення (Bluetooth)
1. Перейдіть на сторінку "Підключення"
2. Переконайтеся що BMS увімкнений та поблизу
3. Натисніть "Шукати BMS"
4. Оберіть ваш пристрій зі списку (зазвичай починається з "xiaoxiang" або "JBD")
5. Дозвольте браузеру підключитися до Bluetooth
6. Дані з'являться на головній сторінці в реальному часі
Підтримувані пристрої
JBD BMS SP14S004 (14S Li-ion)
ESP32-C3 Super Mini
Сумісні JBD/Xiaoxiang BMS з Bluetooth
Браузери: Chrome, Edge, Opera (з підтримкою Web Bluetooth)
Технічні характеристики
Протокол: Xiaoxiang BMS Protocol
Bluetooth: BLE 4.0+
Wi-Fi: 2.4GHz (для ESP32)
Інтервал оновлення: 2 секунди
Підтримка: до 14S конфігурація
Паралельне підключення: так
Корисні поради
Для Wi-Fi підключення ESP32 має бути в одній мережі з вашим пристроєм
Пряме Bluetooth підключення працює тільки в Chrome/Edge/Opera
Демо режим дозволяє протестувати інтерфейс без реальних пристроїв
Використовуйте групи для організації декількох пристроїв
Історія зберігає до 500 останніх записів
При паралельному підключенні напруга усереднюється, струми сумуються
